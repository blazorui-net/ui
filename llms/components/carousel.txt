# Carousel Component

Slideshow component for cycling through content with navigation controls.

---

## Overview

The Carousel component provides a slideshow for cycling through images or content. It includes previous/next navigation buttons and optional loop functionality.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Carousel
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Carousel

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Carousel
```

---

## API Reference

### Main Component: `<Carousel>`

Root carousel container.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Orientation` | `CarouselOrientation` | `Horizontal` | Slide direction |
| `Loop` | `bool` | `false` | Enable infinite looping |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Content, Previous, Next |

### Sub-components

#### `<CarouselContent>`

Container for carousel items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | CarouselItem elements |

#### `<CarouselItem>`

Individual slide wrapper.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Slide content |

#### `<CarouselPrevious>`

Previous slide navigation button.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |

#### `<CarouselNext>`

Next slide navigation button.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |

### Enum: `CarouselOrientation`

| Value | Description |
|-------|-------------|
| `Horizontal` | Slides move left/right |
| `Vertical` | Slides move up/down |

## Basic Usage

```razor
<Carousel Class="w-full max-w-xs">
    <CarouselContent>
        @for (int i = 1; i <= 5; i++)
        {
            <CarouselItem>
                <div class="p-1">
                    <Card>
                        <CardContent Class="flex aspect-square items-center justify-center p-6">
                            <span class="text-4xl font-semibold">@i</span>
                        </CardContent>
                    </Card>
                </div>
            </CarouselItem>
        }
    </CarouselContent>
    <CarouselPrevious />
    <CarouselNext />
</Carousel>
```

## Examples

### Example 1: Image Gallery

```razor
<Carousel Class="w-full max-w-lg">
    <CarouselContent>
        @foreach (var image in images)
        {
            <CarouselItem>
                <img src="@image.Url" alt="@image.Alt" class="rounded-lg" />
            </CarouselItem>
        }
    </CarouselContent>
    <CarouselPrevious />
    <CarouselNext />
</Carousel>

@code {
    private List<(string Url, string Alt)> images = new()
    {
        ("/images/photo1.jpg", "Photo 1"),
        ("/images/photo2.jpg", "Photo 2"),
        ("/images/photo3.jpg", "Photo 3")
    };
}
```

### Example 2: With Loop

```razor
<Carousel Loop="true" Class="w-full max-w-xs">
    <CarouselContent>
        @for (int i = 1; i <= 3; i++)
        {
            <CarouselItem>
                <Card>
                    <CardContent Class="flex aspect-square items-center justify-center bg-primary/10">
                        <span class="text-4xl font-semibold">@i</span>
                    </CardContent>
                </Card>
            </CarouselItem>
        }
    </CarouselContent>
    <CarouselPrevious />
    <CarouselNext />
</Carousel>
```

### Example 3: Product Cards

```razor
<Carousel Class="w-full">

                    </CardContent>
                </Card>

    <CarouselPrevious />
    <CarouselNext />
</Carousel>
```

## Behavior

- Navigation buttons disable at boundaries (unless Loop is enabled)
- Smooth slide transitions
- Supports touch/swipe on mobile devices
- Maintains slide position when resized

---

## Accessibility

- Previous/Next buttons have aria-label attributes
- Slides can be navigated with keyboard
- Focus management between slides
- Screen reader announcements for slide changes
