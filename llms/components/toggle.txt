# Toggle Component

Two-state button for toggleable options.

---

## Overview

The Toggle component is a button that can be switched between pressed and unpressed states. It's useful for options that can be turned on or off, like bold formatting or favorite status.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Toggle
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Toggle

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Toggle
```

---

## API Reference

### Main Component: `<Toggle>`

Two-state toggle button.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Pressed` | `bool` | `false` | Controlled pressed state |
| `PressedChanged` | `EventCallback<bool>` | - | State change callback |
| `DefaultPressed` | `bool` | `false` | Initial pressed state |
| `Variant` | `ToggleVariant` | `Default` | Visual style (Default, Outline) |
| `Size` | `ToggleSize` | `Default` | Size variant |
| `Disabled` | `bool` | `false` | Disables the toggle |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Toggle content |

### Enums

#### `ToggleVariant`

| Value | Description |
|-------|-------------|
| `Default` | Standard toggle style |
| `Outline` | Bordered toggle style |

#### `ToggleSize`

| Value | Description |
|-------|-------------|
| `Default` | Standard size |
| `Small` | Compact size |
| `Large` | Larger size |

## Basic Usage

```razor
<Toggle>
    <LucideIcon Name="bold" Size="16" />
</Toggle>
```

---

## Examples

### Example 1: Text Formatting Toggles

```razor
<div class="flex gap-2">
    <Toggle @bind-Pressed="isBold" aria-label="Toggle bold">
        <LucideIcon Name="bold" Size="16" />
    </Toggle>
    <Toggle @bind-Pressed="isItalic" aria-label="Toggle italic">
        <LucideIcon Name="italic" Size="16" />
    </Toggle>
    <Toggle @bind-Pressed="isUnderline" aria-label="Toggle underline">
        <LucideIcon Name="underline" Size="16" />
    </Toggle>
</div>

@code {
    private bool isBold = false;
    private bool isItalic = false;
    private bool isUnderline = false;
}
```

### Example 2: Outline Variant

```razor
<Toggle Variant="ToggleVariant.Outline" @bind-Pressed="isActive">
    <LucideIcon Name="star" Size="16" />
    <span class="ml-2">Favorite</span>
</Toggle>
```

### Example 3: Different Sizes

```razor
<Toggle Size="ToggleSize.Small">
    <LucideIcon Name="check" Size="14" />
</Toggle>

<Toggle Size="ToggleSize.Default">
    <LucideIcon Name="check" Size="16" />
</Toggle>

<Toggle Size="ToggleSize.Large">
    <LucideIcon Name="check" Size="18" />
</Toggle>
```

### Example 4: Disabled State

```razor
<Toggle Disabled="true">
    <LucideIcon Name="lock" Size="16" />
</Toggle>
```

## Styling

The toggle changes appearance when pressed:
- Default: Semi-transparent background when not pressed
- Pressed: Accented background to indicate active state
- Disabled: Reduced opacity, non-interactive

---

## Accessibility

- Uses `aria-pressed` attribute to indicate state
- Keyboard accessible with Space/Enter keys
- Should include `aria-label` when content is icon-only
