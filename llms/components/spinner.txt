# Spinner Component

A loading spinner that indicates an ongoing process with indeterminate duration.

---

## Overview

The Spinner component provides a visual loading indicator for operations where the progress cannot be determined. It's commonly used during data fetching, form submissions, and page transitions.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Spinner
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Spinner

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Spinner
```

---

## API Reference

### Main Component: `<Spinner>`

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Size` | `SpinnerSize` | `Default` | Size variant (Small, Default, Large) |
| `Class` | `string?` | `null` | Additional CSS classes |

### Enum: `SpinnerSize`

| Value | Size | Description |
|-------|------|-------------|
| `Small` | 16x16 | Inline loading indicators |
| `Default` | 24x24 | Standard loading state |
| `Large` | 32x32 | Full-page or prominent loading |

## Examples

### Example 1: Size Variants

```razor
<div class="flex items-center gap-4">
    <Spinner Size="SpinnerSize.Small" />
    <Spinner Size="SpinnerSize.Default" />
    <Spinner Size="SpinnerSize.Large" />
</div>
```

### Example 2: Button with Loading State

```razor
<Button Disabled="@isLoading" OnClick="HandleClick">
    @if (isLoading)
    {
        <Spinner Size="SpinnerSize.Small" Class="mr-2" />
        <span>Loading...</span>
    }
    else
    {
        <span>Submit</span>
    }
</Button>

@code {
    private bool isLoading;

    private async Task HandleClick()
    {
        isLoading = true;
        await Task.Delay(2000);
        isLoading = false;
    }
}
```

### Example 3: Centered Loading State

```razor
<div class="flex items-center justify-center h-32">
    <Spinner Size="SpinnerSize.Large" />
</div>
```

### Example 4: With Loading Text

```razor
<div class="flex items-center gap-2">
    <Spinner />
    <span class="text-muted-foreground">Loading data...</span>
</div>
```

### Example 5: Full Page Loading

```razor
@if (isPageLoading)
{
    <div class="fixed inset-0 flex items-center justify-center bg-background/80">
        <div class="flex flex-col items-center gap-2">
            <Spinner Size="SpinnerSize.Large" />
            <span class="text-sm text-muted-foreground">Loading...</span>
        </div>
    </div>
}
```

## Accessibility

- Uses `role="status"` for screen reader announcements
- Includes `aria-label="Loading"` for accessibility
- Visible animation provides visual feedback
- Can be combined with text for additional context

---

## Common Patterns

### Conditional Loading

```razor
@if (isLoading)
{
    <Spinner />
}
else
{
    <DataDisplay Data="@data" />
}
```

### Inline Loading

```razor
<span>
    Saving
    <Spinner Size="SpinnerSize.Small" Class="inline ml-1" />
</span>
```

### Card Loading State

```razor
<Card>
    <CardContent class="flex items-center justify-center h-48">
        @if (isLoading)
        {
            <Spinner />
        }
        else
        {
            <!-- Content -->
        }
    </CardContent>
</Card>
```
