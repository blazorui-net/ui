# Progress Component

Displays a progress bar indicating completion percentage of a task.

---

## Overview

The Progress component provides a visual indicator of progress toward completion. It's commonly used for file uploads, form completion, loading states, and any operation with measurable progress.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Progress
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Progress

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Progress
```

---

## API Reference

### Main Component: `<Progress>`

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `double` | `0` | Current progress value |
| `Max` | `double` | `100` | Maximum value (100%) |
| `Class` | `string?` | `null` | Additional CSS classes for container |
| `IndicatorClass` | `string?` | `null` | Additional CSS classes for indicator |

---

## Basic Usage

```razor
<Progress Value="50" />
```

## Examples

### Example 1: Basic Progress

```razor
<Progress Value="33" />
```

### Example 2: Custom Maximum

```razor
<Progress Value="3" Max="10" />
```

### Example 3: With Label

```razor
<div class="space-y-2">
    <div class="flex justify-between text-sm">
        <span>Uploading...</span>
        <span>@progress%</span>
    </div>
    <Progress Value="@progress" />
</div>

@code {
    private double progress = 66;
}
```

### Example 4: Animated Progress

```razor
<Progress Value="@progress" />
<Button OnClick="StartProgress">Start</Button>

@code {
    private double progress = 0;
    private Timer? timer;

    private void StartProgress()
    {
        progress = 0;
        timer = new Timer(_ =>
        {
            progress += 10;
            if (progress >= 100)
            {
                timer?.Dispose();
            }
            InvokeAsync(StateHasChanged);
        }, null, 0, 500);
    }
}
```

### Example 5: Different Sizes

```razor
<div class="space-y-4">
    <Progress Value="60" Class="h-1" />
    <Progress Value="60" Class="h-2" />
    <Progress Value="60" Class="h-4" />
</div>
```

### Example 6: Custom Colors

```razor
<Progress Value="75" IndicatorClass="bg-green-500" />
<Progress Value="25" IndicatorClass="bg-yellow-500" />
<Progress Value="90" IndicatorClass="bg-red-500" />
```

## Styling

### Custom Height

```razor
<Progress Value="50" Class="h-4" />  <!-- Thick -->
```

### Custom Indicator Color

```razor
<Progress Value="50" IndicatorClass="bg-blue-500" />
```

### Rounded Corners

```razor
<Progress Value="50" Class="rounded-full" IndicatorClass="rounded-full" />
```

## Accessibility

- Uses native `<progress>` semantics via ARIA attributes
- `role="progressbar"` for screen readers
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` attributes
- Visible progress indicator with smooth transitions

## Common Patterns

### File Upload Progress

```razor
<div class="space-y-2">
    <div class="flex items-center gap-2">
        <span class="text-sm font-medium">document.pdf</span>
        <span class="text-sm text-muted-foreground">@uploadProgress%</span>
    </div>
    <Progress Value="@uploadProgress" />
</div>
```

### Step Progress

```razor
<Progress Value="@currentStep" Max="@totalSteps" />
<p class="text-sm text-muted-foreground">
    Step @currentStep of @totalSteps
</p>

@code {
    private int currentStep = 2;
    private int totalSteps = 5;
}
```
