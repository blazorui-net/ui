# Breadcrumb Component

Navigation trail showing the current page's location within a hierarchical structure.

---

## Overview

The Breadcrumb component displays a navigation trail that helps users understand their current location and navigate back through the hierarchy. It follows accessibility guidelines with proper ARIA attributes.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Breadcrumb
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Breadcrumb

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Breadcrumb
```

## API Reference

### Main Component: `<Breadcrumb>`

Container for the breadcrumb navigation.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | BreadcrumbList content |

### Sub-components

#### `<BreadcrumbList>`

Ordered list container for breadcrumb items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | BreadcrumbItem elements |

#### `<BreadcrumbItem>`

Individual breadcrumb item wrapper.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Link or page content |

#### `<BreadcrumbLink>`

Link to a parent page in the hierarchy.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Href` | `string?` | `null` | Navigation URL |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Link text |

#### `<BreadcrumbPage>`

Current page (non-clickable).

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Page text |

#### `<BreadcrumbSeparator>`

Visual separator between items (default: /).

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Custom separator content |

#### `<BreadcrumbEllipsis>`

Ellipsis indicator for collapsed items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |

## Basic Usage

```razor
<Breadcrumb>
    <BreadcrumbList>
        <BreadcrumbItem>
            <BreadcrumbLink Href="/">Home</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
            <BreadcrumbLink Href="/components">Components</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
            <BreadcrumbPage>Breadcrumb</BreadcrumbPage>
        </BreadcrumbItem>
    </BreadcrumbList>
</Breadcrumb>
```

## Accessibility

- Uses `<nav aria-label="breadcrumb">` for screen readers
- Uses `<ol>` ordered list for proper semantics
- Current page has `aria-current="page"`
- Separators are hidden from screen readers with `aria-hidden`
