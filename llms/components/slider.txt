# Slider Component

Range input for selecting numeric values within a specified range.

---

## Overview

The Slider component provides a draggable control for selecting a value from a continuous range. It supports keyboard navigation, custom ranges, and step values.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Slider
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Slider

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Slider
```

---

## API Reference

### Main Component: `<Slider>`

Range slider input control.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `double` | `0` | Current value |
| `ValueChanged` | `EventCallback<double>` | - | Value change callback |
| `Min` | `double` | `0` | Minimum value |
| `Max` | `double` | `100` | Maximum value |
| `Step` | `double` | `1` | Step increment |
| `Orientation` | `SliderOrientation` | `Horizontal` | Orientation |
| `Disabled` | `bool` | `false` | Disables the slider |
| `Class` | `string?` | `null` | Additional CSS classes |

### Enum: `SliderOrientation`

| Value | Description |
|-------|-------------|
| `Horizontal` | Standard horizontal slider |
| `Vertical` | Vertical slider |

## Basic Usage

```razor
<Slider @bind-Value="volume" />
<p>Volume: @volume</p>

@code {
    private double volume = 50;
}
```

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `ArrowRight` / `ArrowUp` | Increase by step |
| `ArrowLeft` / `ArrowDown` | Decrease by step |
| `Home` | Set to minimum |
| `End` | Set to maximum |
| `PageUp` | Increase by large step (10x) |
| `PageDown` | Decrease by large step (10x) |

---

## Styling

- Track shows the full range
- Filled portion indicates current value
- Thumb is draggable and focusable
- Disabled state reduces opacity

---

## Accessibility

- Uses `role="slider"` with proper ARIA attributes
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` are set
- Keyboard accessible with standard slider keys
- Focus indicator on the thumb
