# Toggle Group Component

Group of toggles with single or multiple selection support.

---

## Overview

The Toggle Group component allows grouping multiple toggles together with either single (radio-like) or multiple (checkbox-like) selection behavior. Useful for toolbar groups and option selectors.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Toggle
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Toggle

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Toggle
```

## API Reference

### Main Component: `<ToggleGroup>`

Container for grouped toggle items.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Type` | `ToggleGroupType` | `Single` | Selection mode (Single, Multiple) |
| `Value` | `string?` | `null` | Selected value (single mode) |
| `ValueChanged` | `EventCallback<string?>` | - | Value change callback (single mode) |
| `Values` | `HashSet<string>?` | `null` | Selected values (multiple mode) |
| `ValuesChanged` | `EventCallback<HashSet<string>>` | - | Values change callback (multiple mode) |
| `DefaultValue` | `string?` | `null` | Initial value (single mode) |
| `DefaultValues` | `HashSet<string>?` | `null` | Initial values (multiple mode) |
| `Disabled` | `bool` | `false` | Disables all items |
| `Variant` | `ToggleVariant` | `Default` | Visual style |
| `Size` | `ToggleSize` | `Default` | Size variant |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | ToggleGroupItem elements |

### Sub-component: `<ToggleGroupItem>`

Individual item within the group.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `string` | - | **Required.** Item's value |
| `Disabled` | `bool` | `false` | Disables this item |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Item content |

### Enums

#### `ToggleGroupType`

| Value | Description |
|-------|-------------|
| `Single` | Only one item can be selected (radio behavior) |
| `Multiple` | Multiple items can be selected (checkbox behavior) |

## Basic Usage

```razor
<ToggleGroup Type="ToggleGroupType.Single" @bind-Value="alignment">
    <ToggleGroupItem Value="left">
        <LucideIcon Name="align-left" Size="16" />
    </ToggleGroupItem>
    <ToggleGroupItem Value="center">
        <LucideIcon Name="align-center" Size="16" />
    </ToggleGroupItem>
    <ToggleGroupItem Value="right">
        <LucideIcon Name="align-right" Size="16" />
    </ToggleGroupItem>
</ToggleGroup>

@code {
    private string? alignment = "left";
}
```

## Examples

### Example 1: Single Selection (Default)

```razor
<ToggleGroup @bind-Value="selectedView">
    <ToggleGroupItem Value="grid">
        <LucideIcon Name="layout-grid" Size="16" />
    </ToggleGroupItem>
    <ToggleGroupItem Value="list">
        <LucideIcon Name="list" Size="16" />
    </ToggleGroupItem>
</ToggleGroup>

<p>Selected: @selectedView</p>

@code {
    private string? selectedView = "grid";
}
```

### Example 2: Multiple Selection

```razor
<ToggleGroup Type="ToggleGroupType.Multiple" @bind-Values="formatting">
    <ToggleGroupItem Value="bold">
        <LucideIcon Name="bold" Size="16" />
    </ToggleGroupItem>
    <ToggleGroupItem Value="italic">
        <LucideIcon Name="italic" Size="16" />
    </ToggleGroupItem>
    <ToggleGroupItem Value="underline">
        <LucideIcon Name="underline" Size="16" />
    </ToggleGroupItem>
</ToggleGroup>

<p>Selected: @string.Join(", ", formatting)</p>

@code {
    private HashSet<string> formatting = new() { "bold" };
}
```

### Example 3: Outline Variant

```razor
<ToggleGroup Variant="ToggleVariant.Outline" @bind-Value="size">
    <ToggleGroupItem Value="sm">S</ToggleGroupItem>
    <ToggleGroupItem Value="md">M</ToggleGroupItem>
    <ToggleGroupItem Value="lg">L</ToggleGroupItem>
    <ToggleGroupItem Value="xl">XL</ToggleGroupItem>
</ToggleGroup>
```

### Example 4: With Disabled Items

```razor
<ToggleGroup @bind-Value="option">
    <ToggleGroupItem Value="a">Option A</ToggleGroupItem>
    <ToggleGroupItem Value="b" Disabled="true">Option B</ToggleGroupItem>
    <ToggleGroupItem Value="c">Option C</ToggleGroupItem>
</ToggleGroup>
```

## Styling

- Items are visually grouped with connected styling
- Selected items have accented background
- Items share variant and size from parent ToggleGroup
- Disabled items have reduced opacity

---

## Accessibility

- Uses `role="group"` for grouping
- Items have `aria-pressed` attribute
- Keyboard navigation with arrow keys
- Focus management within the group
