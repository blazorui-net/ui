# Date Picker Component

A date picker component with a popover calendar for selecting dates.

---

## Overview

The DatePicker component combines a button trigger with a popover calendar, providing a compact way to select dates. It includes all Calendar features plus customizable date formatting and placeholder text.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.DatePicker
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/DatePicker

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.DatePicker
```

---

## API Reference

### Main Component: `<DatePicker>`

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `DateTime?` | `null` | Currently selected date |
| `ValueChanged` | `EventCallback<DateTime?>` | - | Callback when value changes |
| `DateFormat` | `string` | `"MMMM d, yyyy"` | Format string for displaying date |
| `Placeholder` | `string` | `"Pick a date"` | Text when no date selected |
| `MinDate` | `DateTime?` | `null` | Minimum selectable date |
| `MaxDate` | `DateTime?` | `null` | Maximum selectable date |
| `DisabledDates` | `Func<DateTime, bool>?` | `null` | Function to determine disabled dates |
| `FirstDayOfWeek` | `DayOfWeek` | `Sunday` | Which day starts the week |
| `Disabled` | `bool` | `false` | Whether the picker is disabled |
| `Class` | `string?` | `null` | Additional CSS classes |

```razor
<DatePicker @bind-Value="_selectedDate" />
```

## Examples

### Example 1: Preselected Date

```razor
<DatePicker @bind-Value="_date" />

@code {
    private DateTime? _date = DateTime.Today;
}
```

### Example 2: Custom Placeholder

```razor
<DatePicker @bind-Value="_birthDate" Placeholder="Select birth date" />
```

### Example 3: Custom Date Formats

```razor
<!-- Long format (default) -->
<DatePicker @bind-Value="_date1" DateFormat="MMMM d, yyyy" />
<!-- Displays: January 20, 2025 -->

<!-- Short format -->
<DatePicker @bind-Value="_date2" DateFormat="MM/dd/yyyy" />
<!-- Displays: 01/20/2025 -->

<!-- ISO format -->
<DatePicker @bind-Value="_date3" DateFormat="yyyy-MM-dd" />
<!-- Displays: 2025-01-20 -->

<!-- European format -->
<DatePicker @bind-Value="_date4" DateFormat="dd/MM/yyyy" />
<!-- Displays: 20/01/2025 -->
```

### Example 4: Date Constraints

```razor
<!-- Future dates only -->
<DatePicker @bind-Value="_futureDate" MinDate="@DateTime.Today" />

<!-- Past dates only -->
<DatePicker @bind-Value="_pastDate" MaxDate="@DateTime.Today" />

<!-- Next 30 days only -->
<DatePicker @bind-Value="_rangeDate"
            MinDate="@DateTime.Today"
            MaxDate="@DateTime.Today.AddDays(30)" />
```

### Example 5: Disabled Weekends

```razor
<DatePicker @bind-Value="_date" DisabledDates="@IsWeekend" />

@code {
    private DateTime? _date;

    private bool IsWeekend(DateTime date)
    {
        return date.DayOfWeek == DayOfWeek.Saturday ||
               date.DayOfWeek == DayOfWeek.Sunday;
    }
}
```

### Example 6: Week Start Day

```razor
<div class="flex gap-4">
    <div>
        <p class="text-sm mb-2">Sunday (default):</p>
        <DatePicker @bind-Value="_date1" FirstDayOfWeek="DayOfWeek.Sunday" />
    </div>
    <div>
        <p class="text-sm mb-2">Monday:</p>
        <DatePicker @bind-Value="_date2" FirstDayOfWeek="DayOfWeek.Monday" />
    </div>
</div>
```

### Example 7: Disabled State

```razor
<DatePicker @bind-Value="_date" Disabled="true" />
```

## Date Format Patterns

Common format patterns for the `DateFormat` parameter:

| Pattern | Example | Description |
|---------|---------|-------------|
| `MMMM d, yyyy` | January 20, 2025 | Full month name |
| `MMM d, yyyy` | Jan 20, 2025 | Abbreviated month |
| `MM/dd/yyyy` | 01/20/2025 | US short format |
| `dd/MM/yyyy` | 20/01/2025 | European format |
| `yyyy-MM-dd` | 2025-01-20 | ISO format |
| `d MMMM yyyy` | 20 January 2025 | Day first, full month |
| `ddd, MMM d` | Mon, Jan 20 | Day name, short |

## Common Patterns

### Form Integration

```razor
<Field>
    <Label>Date of Birth</Label>
    <DatePicker @bind-Value="_birthDate"
                MaxDate="@DateTime.Today"
                Placeholder="Select your birth date" />
</Field>
```

### Event Date Range

```razor
<div class="flex gap-4">
    <Field>
        <Label>Start Date</Label>
        <DatePicker @bind-Value="_startDate"
                    MinDate="@DateTime.Today" />
    </Field>
    <Field>
        <Label>End Date</Label>
        <DatePicker @bind-Value="_endDate"
                    MinDate="@(_startDate ?? DateTime.Today)" />
    </Field>
</div>

@code {
    private DateTime? _startDate;
    private DateTime? _endDate;
}
```

### Appointment Booking

```razor
<DatePicker @bind-Value="_appointmentDate"
            MinDate="@DateTime.Today"
            DisabledDates="@IsUnavailable"
            Placeholder="Select appointment date" />

@code {
    private DateTime? _appointmentDate;

    private bool IsUnavailable(DateTime date)
    {
        // Disable weekends and holidays
        return date.DayOfWeek == DayOfWeek.Sunday || IsHoliday(date);
    }
}
```
