# Context Menu Component

Right-click menu with customizable items, shortcuts, and labels.

---

## Overview

The Context Menu component displays a menu triggered by right-clicking on a designated area. It supports items, separators, labels, and keyboard shortcuts similar to native context menus.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.ContextMenu
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/ContextMenu

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.ContextMenu
```

## API Reference

### Main Component: `<ContextMenu>`

Root container managing menu state.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Trigger and content |

### Sub-components

#### `<ContextMenuTrigger>`

The area that triggers the context menu on right-click.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Trigger content |

#### `<ContextMenuContent>`

The menu popup container.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Menu items |

#### `<ContextMenuItem>`

Clickable menu item.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `OnClick` | `EventCallback` | - | Click handler |
| `Disabled` | `bool` | `false` | Disables the item |
| `Inset` | `bool` | `false` | Adds left padding |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Item content |

#### `<ContextMenuLabel>`

Non-interactive section label.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Inset` | `bool` | `false` | Adds left padding |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Label text |

#### `<ContextMenuSeparator>`

Visual divider between items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |

#### `<ContextMenuShortcut>`

Keyboard shortcut hint.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Shortcut text |

## Basic Usage

```razor
<ContextMenu>
    <ContextMenuTrigger>
        <div class="h-[150px] w-[300px] border border-dashed flex items-center justify-center">
            Right click here
        </div>
    </ContextMenuTrigger>
    <ContextMenuContent>
        <ContextMenuItem OnClick="@(() => HandleAction("Back"))">
            Back
            <ContextMenuShortcut>Ctrl+[</ContextMenuShortcut>
        </ContextMenuItem>
        <ContextMenuItem OnClick="@(() => HandleAction("Forward"))" Disabled="true">
            Forward
            <ContextMenuShortcut>Ctrl+]</ContextMenuShortcut>
        </ContextMenuItem>
        <ContextMenuSeparator />
        <ContextMenuItem OnClick="@(() => HandleAction("Reload"))">
            Reload
            <ContextMenuShortcut>Ctrl+R</ContextMenuShortcut>
        </ContextMenuItem>
    </ContextMenuContent>
</ContextMenu>

@code {
    private void HandleAction(string action)
    {
        Console.WriteLine($"Action: {action}");
    }
}
```

## Examples

### Example 1: With Labels and Groups

```razor
<ContextMenu>
    <ContextMenuTrigger>
        <div class="h-[150px] w-[300px] border">Right click here</div>
    </ContextMenuTrigger>
    <ContextMenuContent>
        <ContextMenuLabel>Edit</ContextMenuLabel>
        <ContextMenuItem>Cut</ContextMenuItem>
        <ContextMenuItem>Copy</ContextMenuItem>
        <ContextMenuItem>Paste</ContextMenuItem>
        <ContextMenuSeparator />
        <ContextMenuLabel>View</ContextMenuLabel>
        <ContextMenuItem>Zoom In</ContextMenuItem>
        <ContextMenuItem>Zoom Out</ContextMenuItem>
    </ContextMenuContent>
</ContextMenu>
```

### Example 2: File Manager Context Menu

```razor
<ContextMenu>
    <ContextMenuTrigger>
        <div class="p-4 border rounded">My Document.pdf</div>
    </ContextMenuTrigger>
    <ContextMenuContent>
        <ContextMenuItem OnClick="OpenFile">
            Open
            <ContextMenuShortcut>Enter</ContextMenuShortcut>
        </ContextMenuItem>
        <ContextMenuItem OnClick="RenameFile">
            Rename
            <ContextMenuShortcut>F2</ContextMenuShortcut>
        </ContextMenuItem>
        <ContextMenuSeparator />
        <ContextMenuItem OnClick="DeleteFile" Class="text-destructive">
            Delete
            <ContextMenuShortcut>Del</ContextMenuShortcut>
        </ContextMenuItem>
    </ContextMenuContent>
</ContextMenu>
```

## Behavior

- Menu appears at the cursor position on right-click
- Menu closes when clicking outside or pressing Escape
- Disabled items cannot be clicked but remain visible
- Uses Portal pattern for proper z-index stacking

---

## Accessibility

- Follows WAI-ARIA menu pattern
- Keyboard navigation with arrow keys
- Items are focusable and have appropriate ARIA attributes
- Escape key closes the menu
