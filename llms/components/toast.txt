# Toast Component

A succinct message that is displayed temporarily to notify users of an event.

---

## Overview

The Toast component provides temporary notifications that appear and disappear automatically. It includes a service-based API for programmatic control, multiple positions, variants, and optional action buttons.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Toast
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Toast

## Installation

```bash
dotnet add package BlazorUI.Components
```

### Setup

1. Register the service in `Program.cs`:

```csharp
builder.Services.AddScoped<ToastService>();
```

2. Add `ToastProvider` to your layout (e.g., `MainLayout.razor`):

```razor
@using BlazorUI.Components.Toast

<!-- Your layout content -->
@Body

<ToastProvider Position="ToastPosition.BottomRight" />
```

3. Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Toast
```

## API Reference

### ToastService

Inject `ToastService` to show toasts programmatically.

**Methods:**

| Method | Description |
|--------|-------------|
| `Show(ToastData toast)` | Show a toast with full control |
| `Show(description, title?, variant?, duration?)` | Quick toast |
| `Success(description, title?)` | Show success toast |
| `Error(description, title?)` | Show error toast |
| `Dismiss(id)` | Dismiss specific toast by ID |
| `DismissAll()` | Dismiss all toasts |

### ToastProvider

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Position` | `ToastPosition` | `BottomRight` | Where toasts appear |
| `MaxToasts` | `int` | `5` | Maximum visible toasts |
| `Class` | `string?` | `null` | Additional CSS classes |

### ToastData

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `Id` | `string` | Auto-generated | Unique identifier |
| `Title` | `string?` | `null` | Optional title |
| `Description` | `string?` | `null` | Main message |
| `Variant` | `ToastVariant` | `Default` | Visual style |
| `Duration` | `int` | `5000` | Auto-dismiss time (ms), 0 for no auto-dismiss |
| `ShowClose` | `bool` | `true` | Show close button |
| `ActionText` | `string?` | `null` | Optional action button text |
| `OnAction` | `Action?` | `null` | Action button callback |

### Enums

**ToastVariant:**
| Value | Description |
|-------|-------------|
| `Default` | Standard styling |
| `Destructive` | Error/warning styling |

**ToastPosition:**
| Value | Description |
|-------|-------------|
| `TopLeft` | Top-left corner |
| `TopCenter` | Top-center |
| `TopRight` | Top-right corner |
| `BottomLeft` | Bottom-left corner |
| `BottomCenter` | Bottom-center |
| `BottomRight` | Bottom-right corner (default) |

## Basic Usage

```razor
@inject ToastService ToastService

<Button OnClick="ShowToast">Show Toast</Button>

@code {
    private void ShowToast()
    {
        ToastService.Show("Your message has been sent.");
    }
}
```

## Examples

### Example 1: Simple Toast

```razor
ToastService.Show("Operation completed successfully.");
```

### Example 2: Toast with Title

```razor
ToastService.Show("Friday, February 10, 2023 at 5:57 PM", "Scheduled: Catch up");
```

### Example 3: Success Toast

```razor
ToastService.Success("Your changes have been saved.", "Success");
```

### Example 4: Error Toast

```razor
ToastService.Error("There was a problem with your request.", "Error");
```

### Example 5: Toast with Action

```razor
ToastService.Show(new ToastData
{
    Title = "Undo action",
    Description = "Your post has been deleted.",
    ActionText = "Undo",
    OnAction = () => ToastService.Success("Post restored!")
});
```

### Example 6: Custom Duration

```razor
// Quick toast (2 seconds)
ToastService.Show("This disappears quickly!", duration: 2000);

// Long toast (10 seconds)
ToastService.Show("This stays longer.", duration: 10000);

// Persistent toast (no auto-dismiss)
ToastService.Show(new ToastData
{
    Title = "Persistent Toast",
    Description = "Click X to close.",
    Duration = 0
});
```

### Example 7: Multiple Toasts

```razor
ToastService.Show("First notification");
ToastService.Show("Second notification", "Alert");
ToastService.Error("Third notification is an error");
```

### Example 8: Dismiss All

```razor
ToastService.DismissAll();
```

## Accessibility

- Uses `role="status"` for non-critical toasts
- Proper ARIA attributes for screen readers
- Close button is keyboard accessible
- Action buttons are focusable
- Auto-dismiss respects user preferences

## Common Patterns

### Form Submission Feedback

```razor
private async Task HandleSubmit()
{
    try
    {
        await SaveData();
        ToastService.Success("Data saved successfully!");
    }
    catch (Exception ex)
    {
        ToastService.Error(ex.Message, "Save Failed");
    }
}
```

### API Error Handling

```razor
private async Task FetchData()
{
    var response = await Http.GetAsync("/api/data");
    if (!response.IsSuccessStatusCode)
    {
        ToastService.Error("Failed to load data. Please try again.");
    }
}
```
